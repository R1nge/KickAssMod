Use 'Shrug' emotion to trigger a kick  
To change the emotion open `config_folder/KickAssMod.cfg' and change 'KickEmotion' value  

Possible values: 
- Salute
- Thumbs Up
- Think
- No-No
- Play Dead
- Shrug
- Crossed Arms
- Dance
